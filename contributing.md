Sass2Stylus
----------
#[Use it live](http://sass2stylus.com)

### Development
- Clone this repository
- `bundle install`
- `cd front_end`
- `npm install -g roots`
- `roots compile`
- `cd ..`
- `rackup config.ru`

### Deployment to Heroku
- `heroku create`
- `./deploy.sh`
